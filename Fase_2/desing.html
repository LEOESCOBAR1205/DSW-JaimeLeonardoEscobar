Proposito del sitio

Este tiene como proposito el aprendizaje del diseño de paginas web, en esta ocasion vamos a profundizar sobre la estructura de las listan de enlace y flexbox. Dando a concoer sus estructuras y composicion.

FLEXBOX

Flexbox es un sistema de elementos flexibles que llega con la idea de olvidar estos mecanismos y acostumbrarnos a una mecánica más potente, limpia y personalizable, en la que los elementos HTML se adaptan y colocan automáticamente y es más fácil personalizar los diseños. Está especialmente diseñado para crear, mediante CSS, estructuras de una sóla dimensión.

Aubry, C. (2014). HTML5 y CSS3: para sitios con diseño web responsive. Ediciones ENI.

LISTAS DE ENLACE

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.
Los tipos de listas en HTML son los siguientes:
Listas ordenadas
Listas desordenadas
Listas de definiciones
